class ral::frame::BlazingHostTable¶

A class that represents the BlazingTable store in host memory. This implementation uses only raw allocations, ColumnTransports and chunked_column_infos that represent a BlazingTable. The reference to implement this class was based on the way how BlazingTable objects are send/received by the communication library.
