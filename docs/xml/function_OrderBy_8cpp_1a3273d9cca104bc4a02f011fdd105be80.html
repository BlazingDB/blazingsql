int64_t ral::operators::determine_local_limit(Context*, int64_t, cudf::size_type)¶

In a distributed context, this function determines what the limit would be for this local node. It does this be distributing and collecting the total number of rows in the table. Then knowing which node index this local node is, it can calculate how many rows are ahead of the ones in this partition.
